## Technologies

- **Frontend**: React + Vite + Tailwind CSS + shadcn/ui
- **Backend**: PHP + MySQL
- **Auth**: JWT tokens

## Installation pour regarder le code en local

### Backend (PHP)
```bash
cd api
php -S localhost:8000 # Sur linux (sinon faire avec wsl)
```

### Frontend (React)
```bash
cd react
npm install
npm run dev
```

## Fonctionnalités

- Inscription / Connexion
- Créer, modifier, supprimer des tâches
- Créer des catégories pour organiser
- Filtrer par statut et catégorie
- Trier les tâches
- Modifier son profil (nom, email, mot de passe)
- Supprimer son compte
